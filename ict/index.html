Base Structure

Tag: !DOCTYPE, html, head, meta, title, body

Structure

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page title</title>
</head>
<body>
 ...... 
</body>
</html>

Text Formatting

Tag:

  • Bold : strong, b
  • Itallic : em, i
  • Underlined : u
  • Line through : del
  • Subscript : sub
  • Superscript : sup

Example

<p> This is <b>Bold text</b></p>
<p> This is <strong>Bold text</strong></p>
<p> This is <i>Itallic text</i></p>
<p> This is <em>Itallic text</em></p>
<p> This is <u>Underlined text</u></p>
<p> This is <del>Line through text</del></p>
<p> This is <sub>Subscript</sub></p>
<p> This is <sup>Superscript</sup></p>

Output

This is Bold text

This is Bold text

This is Itallic text

This is Itallic text

This is Underlined text

This is Line through text

This is Subscript

This is Superscript


List

Tag:

  • Ordered list : ol
  • Unordered list : ul
  • List item : li

Attributes:

  • Ordered list :
    • type="" : 1, i, I, a, A
    • start="" : starting number for the list
  • Unordered list :
    • type="" : square, circle, disc

Example [ordered list]

<ol>This is an Ordered list
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

Output

    This is an Ordered list
  1. Item 1
  2. Item 2
  3. Item 3

Example [unordered list]

<ul>This is an Unordered list
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Output

    This is an Unordered list
  • Item 1
  • Item 2
  • Item 3

Anchor

Tag: a

Attributes:

  • href="" : hyperlink / URL
  • target="" : _blank

Example

<a href="https://google.com/">Click Here</a>

Output

Click Here

Meta

Tag: meta

Note: Meta tag doesn’t show up on website, it only contains informations related to the website inside head tag! Meta tag contains informations and instructions about webpage.

Attributes:

  • charset="" : utf-8

Example

<meta charset="UTF-8">

Image

Tag: img

Attributes:

  • src="" : Image link
  • alt="" : Alternate Name
  • title="" : Title for tooltip
  • width="" : Width in px
  • height="" : Height in px

Example

<img src="https://i.pinimg.com/originals/a5/5c/07/a55c077d992aeab224bb9bfd9d0b44bf.png" width="200px" alt="Site Logo">

Output


Div

Tag: div

Note: Divs are just containers that contains a block of codes!

Example

<div>
  <p>This is a paragraph inside a div</p>
</div>

Output

Span

Tag: span

Note: Spans are containers like div but they are inline element while divs are block level element!

Example

<p>This is a <span>Span</span> inside a p tag</p>

Output

This is a Span inside a p tag


Table

Tag:

  • Table : table
  • Table head : thead
  • Table body : tbody
  • Table footer : tfooter
  • Table row : tr
  • Table data : td, th

Attributes:

  • Table :
    • border="" : 0(default) = none; 1,2,3.... = bordered
    • cellspacing="" : spacing between cells (0,1,2,3....)
    • cellpadding="" : padding inside cells (0,1,2,3....)
  • Table Data :
    • rowspan="" : number of rows (1,2,3....)
    • colspan="" : number of columns (1,2,3....)
    • align="" : horizontal alignment (left, center, right)
    • valign="" : vertical alignment (top, middle, center)

Example

<table>
  <tr>
    <th>Id.</th>
    <th>Name</th>
    <th>Country</th>
  </tr>
  <tr>
    <td>01</td>
    <td>John Doe</td>
    <td>USA</td>
  </tr>
  <tr>
    <td>02</td>
    <td>Dany Krossing</td>
    <td>Poland</td>
  </tr>
</table>

Output

Id. Name Country
01 John Doe USA
02 Dany Krossing Poland


Example for border=""

<table border="1">
......
</tabe>

Output

Id. Name Country
01 John Doe USA
02 Dany Krossing Poland

Example for cellspacing=""

<table border="1" cellspacing="0">
......
</tabe>

Output

Id. Name Country
01 John Doe USA
02 Dany Krossing Poland

Example for cellpadding=""

<table border="1" cellpadding="15">
......
</tabe>

Output

Id. Name Country
01 John Doe USA
02 Dany Krossing Poland
