HTML Cheat Sheet for HSC

by Hind Biswas

Base Structure

Tag: !DOCTYPE, html, head, meta, title, body

Structure

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page title</title>
</head>
<body>
  
</body>
</html>

Paragraph

Tag: p

Example

<p>This is a paragraph</p>

Heading

Tag: h1, h2, h3, h4, h5, h6

Example

<h1> This is Heading 1 </h1>
<h2> This is Heading 2 </h2>
<h3> This is Heading 3 </h3>
<h4> This is Heading 4 </h4>
<h5> This is Heading 5 </h5>
<h6> This is Heading 6 </h6>

Break

Tag: br

Example

<p>Before break<br>This is after break</p>

Horizontal Break

Tag: hr

Example

<p>Before horizontal break<hr>After horizontal break</p>

Text Formatting

Tag:

  • Bold : strong, b
  • Itallic : em, i
  • Underlined : u
  • Subscript : sub
  • Superscript : sup

Example

<p> This is <b>Bold text</b></p>
<p> This is <strong>Bold text</strong></p>
<p> This is <i>Itallic text</i></p>
<p> This is <em>Itallic text</em></p>
<p> This is <u>Underlined text</u></p>
<p> This is <sub>Subscript</sub></p>
<p> This is <sup>Superscript</sup></p>

List

Tag:

  • Ordered list : ol
  • Unordered list : ul
  • List iten : li

Attributes:

  • Ordered list :
    • type="" : 1, i, I, a, A
    • start="" : starting number for the list
  • Unordered list :
    • type="" : square, circle, disc

Example [ordered list]

<ol>This is an Ordered list
  <li>Item 1</li>
  <li>Item 1</li>
  <li>Item 1</li>
</ol>

Example [unordered list]

<ul>This is an Ordered list
  <li>Item 1</li>
  <li>Item 1</li>
  <li>Item 1</li>
</ul>

Anchor

Tag: a

Attributes:

  • href="" : hyperlink / URL
  • target="" : _blank

Example

<a href="https://google.com/">Click Here</a>

Meta

Tag: meta

Attributes:

  • charset="" : utf-8

Example

<meta charset="UTF-8">

Image

Tag: img

Attributes:

  • src="" : Image link
  • alt="" : Alternate Name
  • title="" : Title for tooltip
  • width="" : Width in px
  • height="" : Height in px

Example

<img src="images/logo.png" width="200px" alt="Site Logo">

Div

Tag: div

Example

<div>
  <p>This is a paragraph inside a div</p>
</div>

Span

Tag: span

Example

<p>This is a <span>Span</span> inside a p tag</p>

List

Tag:

  • Table : table
  • Table head : thead
  • Table body : tbody
  • Table footer : tfooter
  • Table row : tr
  • Table data : td, th

Attributes:

  • Table :
    • border="" : 0(default) = none; 1,2,3.... = bordered
    • cellspacing="" : spacing between cells (0,1,2,3....)
    • cellpadding="" : padding inside cells (0,1,2,3....)
  • Table Data :
    • rowspan="" : number of rows (1,2,3....)
    • colspan="" : number of columns (1,2,3....)

Example

<table>
  <tr>
    <th>Id.</th>
    <th>Name</th>
    <th>Country</th>
  </tr>
  <tr>
    <td>01</td>
    <td>John Doe</td>
    <td>USA</td>
  </tr>
  <tr>
    <td>02</td>
    <td>Dany Krossing</td>
    <td>Poland</td>
  </tr>
</table>

Style

Tag: style

Properties:

  • width height
  • font-family font-weightfont-size
  • margin padding
  • border border-radius
  • float text-align
  • color background

    • Structure

    selector{
  property: value;
}

    Example

    <style>
  div{
    width: 100px;
    height: 100px;
    margin: 10px;
    padding: 10px;
    font-family: sans-serif;
    font-size: 18px;
    border: 2px solid red;
    border-radius: 5px;
    text-align: center;
    color: white;
    background: pink;
 }
</style>
